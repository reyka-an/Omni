# Omni — персональный трекер привычек и жизни

Omni — мобильное приложение для ведения привычек, задач и личной статистики в одном месте.
Все модули независимы и могут быть включены или скрыты пользователем.

## Основные модули

### 1. Привычки (челленджи)

Формат ежедневных челленджей.

Функции:

* Создание привычки с описанием (например: «Читать 1 час»)
* Ежедневная отметка выполнения
* Подсчёт:

    * текущего стрика
    * максимального стрика
* Каждая привычка автоматически добавляется в список дел на день

---

### 2. Список дел

Классический TODO-лист.

Функции:

* Дата
* Время
* Заметки
* Автодобавление задач из привычек

---

### 3. Учёт времени + напоминания

Тайм-трекер для занятий (работа, учёба, спорт и т.д.).

Функции:

* Запуск таймера перед началом занятия
* Фиксация времени начала и окончания
* Подсчёт общего времени за сессию
* Настраиваемые напоминания:

    * например, напоминание размяться каждые 30/60 минут

---

### 4. Настроение

Дневник состояния.

Функции:

* Выбор дня
* Теги настроения и состояний (например: «устал», «радостный», «тревожный»)
* Возможность добавлять свои теги
* Заметки к дню

---

### 5. Параметры тела

Учёт физических показателей.

Функции:

* Вес
* Обхваты (талия, грудь и т.д.)
* Произвольные параметры
* Просмотр статистики и истории изменений

---

### 6. Просмотрено / хочу посмотреть

Библиотека фильмов, сериалов и аниме.

Функции:

* Добавление элементов в список
* Типы:

    * просмотрено
    * брошено
    * отложено
* Теги (например: «с друзьями», «один», «семья»)
* Заметки
* Оценка
* Сортировка по тегам и статусу

---

### 7. Список покупок

Гибкий список покупок.

Функции:

* Теги по магазинам
* Группировка по тегам
* Быстрое добавление позиций

---

## Гибкая система вкладок

Каждый пользователь может:

* включать или отключать любой модуль
* скрывать ненужные вкладки
* настраивать порядок вкладок

Приложение адаптируется под конкретный сценарий использования:

* только привычки
* только задачи и таймер
* полный набор функций

---

## Цель проекта

Создать единое приложение для:

* самодисциплины
* отслеживания прогресса
* управления повседневными делами
* хранения личных данных без перегруза интерфейса
